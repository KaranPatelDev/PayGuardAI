# AI Services

AI-powered features for invoice OCR, follow-up generation, risk assessment, and recovery reporting.

---

<ApiEndpoint
  method="POST"
  path="/api/ai/parse-invoice"
  description="Extract invoice data from a PDF or image file using OCR. Accepts PDF, PNG, JPG, and WebP formats up to 10MB. Automatically matches against existing customers by business name."
  auth={true}
  body={`(multipart/form-data — file upload)

Form fields:
  file: <binary — PDF, PNG, JPG, or WebP>`}

  "invoice_date": "2026-06-30",
  "matched_customer_id": "cust-001"

  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." \\
