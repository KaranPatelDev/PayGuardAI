# Tech Stack

PayGuard AI is built with a curated set of modern, production-grade technologies.

---

- `@radix-ui/react-dialog`, `react-dropdown-menu`, `react-popover`
- `@radix-ui/react-tabs`, `react-accordion`, `react-collapsible`
- `@radix-ui/react-select`, `react-checkbox`, `react-switch`
- `@radix-ui/react-toast`, `react-tooltip`, `react-label`
- `@radix-ui/react-avatar`, `react-separator`, `react-scroll-area`
- And more — see `frontend/package.json` for the full list.

## Authentication & Security

| Technology | Purpose |
|------------|---------|
| **PyJWT** | JWT access token and reset token creation/verification |
| **bcrypt** | Password hashing with auto-generated salts |
| **HS256** | HMAC-SHA256 token signing algorithm |
| **CORS Middleware** | Cross-origin request control (Starlette) |

### Token Configuration

| Parameter | Default | Env Var |
|-----------|---------|---------|
| Algorithm | HS256 | `JWT_ALGORITHM` |
| Access token TTL | 10080 min (7 days) | `JWT_EXPIRE_MINUTES` |
| Reset token TTL | 15 min | Hardcoded |
| Secret key | — | `JWT_SECRET` (required) |

## AI / LLM

| Technology | Purpose |
|------------|---------|
| **Claude Sonnet 4.6** | Primary LLM for follow-up generation, risk scoring, recovery reports |
| **Emergent LLM Key** | Unified API key for Claude access |
| **Rule-based fallback** | Deterministic template engine when AI is unavailable |

### AI Capabilities

| Feature | Description |
|---------|-------------|
| Follow-up generation | Context-aware payment reminder messages |
| Risk scoring | Customer credit risk assessment (0–100) |
| Recovery reports | AR aging analysis and collection strategy |
| Invoice parsing | OCR-like structured data extraction from uploaded files |

## Deployment

| Component | Platform | Notes |
|-----------|----------|-------|
| **Frontend** | Vercel | Static hosting, edge CDN, automatic deploys |
| **Backend** | Railway | Docker-based Python hosting, env var management |
| **Database** | Neon | Serverless PostgreSQL, branch-based workflows |
| **AI API** | Emergent | Claude Sonnet 4.6 access via unified key |
| **Email** | Gmail SMTP | Free tier for transactional email |

### Build & Development

| Command | Location | Purpose |
|---------|----------|---------|
| `yarn start` | `frontend/` | Start React dev server (via CRACO) |
| `yarn build` | `frontend/` | Production build |
| `uvicorn server:app` | `backend/` | Start FastAPI server |
| `python seed.py` | `backend/` | Seed demo data |
