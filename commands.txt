1. PREREQUISITES
-----------------------------------------------
- Python 3.11+      -> python --version
- Node.js 18+       -> node --version
- Yarn 1.x          -> yarn --version
- MongoDB 6+         -> mongod --version

-----------------------------------------------
2. MONGODB SETUP
-----------------------------------------------

# Start MongoDB (default port 27017)
# Windows:
mongod --dbpath "C:\data\db" --bind_ip 127.0.0.1

# macOS (Homebrew):
brew services start mongodb-community

# Linux:
sudo systemctl start mongod

# Verify MongoDB is running:
mongosh --eval "db.runCommand({ ping: 1 })"

-----------------------------------------------

8. ENVIRONMENT VARIABLES REFERENCE
-----------------------------------------------

BACKEND (.env):
  MONGO_URL          MongoDB connection string
  DB_NAME            Database name
  CORS_ORIGINS       Allowed origins (comma-separated)
  EMERGENT_LLM_KEY   AI provider API key
  JWT_SECRET         Secret for JWT signing
  JWT_ALGORITHM      JWT algorithm (default: HS256)
  JWT_EXPIRE_MINUTES Token expiry (default: 10080 = 7 days)
  DEMO_EMAIL         Demo user email (optional, for seed)
  DEMO_PASSWORD      Demo user password (optional, for seed)

FRONTEND (.env):
  REACT_APP_BACKEND_URL   Backend API base URL

-----------------------------------------------
9. COMMON ISSUES
-----------------------------------------------

# If "MONGO_URL not set" error:
  Make sure backend/.env exists with MONGO_URL

# If "JWT_SECRET not set" error:
  Set JWT_SECRET in backend/.env

# If frontend can't reach backend:
  Check REACT_APP_BACKEND_URL in frontend/.env
  Ensure CORS_ORIGINS includes http://localhost:3000

# If port 8000 already in use:
  uvicorn server:app --port 8001
  (update REACT_APP_BACKEND_URL accordingly)

# If yarn install fails:
  yarn install --ignore-engines
